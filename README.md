# Розробка веб-застосунку "Віртуальний фізичний кабінет" засобами low-code платформ

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра (ВВП "ППФК НТУ "ХПІ"")

Керівник роботи - Бабич Олександр Вікторович 

# Завдання до дипломної роботи 

Продукт має являти собою веб-застосунок, який містить велику кількість інтерактивних лабораторних робіт (симуляцій) з фізики, дозволяє обрати тему чи розділ, вибрати і запустити обрану симуляцію. 

# Використані технології 

* WIX, Google Sites
* PhET Interactive Simulations
* LabXchange
* Low-code

# Опублікований продукт

Інсталяційний пакет створеного веб-застосунку можна завантажити з магазину (додам пізніше)
